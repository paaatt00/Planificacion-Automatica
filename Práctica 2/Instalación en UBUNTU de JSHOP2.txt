- Descomprimimos el archivo
- Seguir el manual LEEME.TXT (importante instalar mediante el comando jdk de la línea 5)

- Los scripts .sh son los scripts de consola (jshop2-console) y de ventana (jshop2-gui)

- La carpeta domains viene con dominios y problemas de ejemplo
	
	--> Para crear nosotros un dominio, debemos crear una carpeta como están creadas las otras
	--> Dentro de la carpeta creamos dos scripts: uno con el mismo nombre (dominio) y otro con el nombre del problema (problem).
	--> Para lanzar un problema, debemos poner de comando en la consola y a continuación el nombre del dominio.
		+> Ejemplo: "./jshop2-gui.sh basic" para la interfaz, y +> "./jshop2-console.sh basic" para la consola donde basic es el nombre del dominio
